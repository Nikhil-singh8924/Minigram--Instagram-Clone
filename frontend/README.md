# Minigram
Minigram is a mini social media application similar to Instagram. Users can sign up, log in, create posts with images and captions, like/unlike posts, comment, follow other users, and view a feed of posts from people they follow.

## Features
- User signup and login
- Create posts with image URLs and captions
- Like/unlike posts
- Comment on posts
- Follow/unfollow other users
- Feed page showing posts from followed users
- Profile page showing posts, followers, and following count

## Running the Project

1. Clone the Repository

git clone <your-repo-url>
cd minigram

### Backend
2. Go to the backend folder:

Run these commands

`cd backend`
`npm install`
`node server.js`

Runs on http://localhost:5000

### Frontend
3. Go to the frontend folder:

Run these commands

`cd frontend`        
`npm install`
`npm start`            // Make sure the backend server is running before opening the frontend.

Runs on http://localhost:3000

## Tech Stack 

Frontend: React.js
Backend: Node.js + Express
Database: JSON files (users.json, posts.json)
Routing: React Router
Styling: CSS
